La Hoguerita es una página web desarrollada con HTML, CSS y JavaScript que simula el sitio oficial de un restaurante de brasas.
El usuario puede:

Navegar por la carta del restaurante

Ver platos organizados por categorías

Añadir productos al carrito

Ver el total actualizado

Eliminar productos del carrito

El objetivo del proyecto es practicar diseño web, DOM, eventos, arrays, y gestión dinámica de datos en JavaScript.

🛠️ Tecnologías Utilizadas
HTML5 — estructura del sitio

CSS — estilos, diseño responsive y estética tabernera

JavaScript — lógica del carrito, interacción y render dinámico

📂 Estructura del Proyecto
index.html — página principal con la carta

styles.css — estilos del sitio

app.js — lógica del carrito y renderizado

/img — imágenes de los platos y del restaurante

🍖 Características Principales
Carta dinámica generada desde JavaScript

Carrito interactivo con suma automática

Botones de añadir en cada plato

Diseño responsive para móviles y tablets

Persistencia opcional con LocalStorage

🧩 Cómo Funciona el Carrito
Cada plato tiene un botón “Añadir al carrito”

Al hacer clic, el producto se agrega a un array carrito[]

El carrito se renderiza en pantalla con:

Nombre del plato

Precio

Cantidad

Botón de eliminar

El total se recalcula automáticamente

▶️ Cómo Ejecutar el Proyecto
Descarga o clona el repositorio

Abre index.html en tu navegador

¡Listo! Puedes navegar y añadir platos al carrito
